Q: Will GitHub Actions work immediately after terraform apply?
A: âŒ NO. You must:

   3. Configure GitHub secrets
   Then GitHub Actions will work.

Q: Do I need to SSH to EC2 every time I update code?
A: âŒ NO. Only for the first deployment. After that, GitHub Actions
   handles everything automatically.

Q: What triggers automatic deployment?
A: âœ… git push origin main (or any push to main branch)

Q: Can I skip manual deployment and just use GitHub Actions?
A: âŒ NO. Manual deployment is required once to:
   - Clone the repository
   - Create initial Docker images
   - Set up the app directory structure
   - Verify everything works
   After that, GitHub Actions can update it.

Q: How long does each deployment take?
A:
   - First deployment (manual): 20-30 minutes total
   - Subsequent deployments (automatic): 2-3 minutes

Q: What if I change Terraform configuration?
A: Run `terraform apply` again. This won't affect your application.
   The app keeps running unless you change the EC2 instance.
